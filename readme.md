 ## Golden Store 
 
 E-commerce website built with laravel framework.
 
 ![ ]( /images/user_homepage.png)
 
## Key Features:-

 # Public User:
   * Search Product
   * View Product
   * Create User Account

 # Signin User:
   * Search Product
   * View Product
   * Add To Cart
   * Create Orders review
   * Orders review
 
 # Admin:
  * Add New Category
  * Update Category
  * Delete Category
  * Add New Product.
  * Add Product images
  * Add product attributes 
  * Update Product 
  * Delete Product
  * View orders 
  *Update Order Status
  * View all registerd users.
